# Digital Credentials Portal

## Getting Started

This project consists of a Next.js frontend and a Nest.js backend.

### Prerequisites

- Node.js installed
- npm installed

### Launching the Project

You will need to run the frontend and backend in separate terminal windows.

#### 1. Frontend (Next.js)

```bash
cd frontend
npm run dev
```

The frontend will be available at [http://localhost:3000](http://localhost:3000).

#### 2. Backend (Nest.js)

```bash
cd backend
npm run start:dev
```

The backend will be available at [http://localhost:3001](http://localhost:3001) (default Nest.js port, might vary if configured differently).

## Project Structure

- `/frontend`: Next.js application (App Router)
- `/backend`: Nest.js application
